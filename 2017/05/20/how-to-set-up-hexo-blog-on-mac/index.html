使用Hexo基于Github搭建个人Blog

发表于 2017-05-20 | 分类于 Work

一直想搭建自己的一个博客，但是都比较懒（逃…
折腾了一下Hexo，过程还是很流畅的
Tip, 本文主要适用于mac

Hexo?

Hexo是什么，来看一下官网的解释

Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other languages) and Hexo generates static files with a beautiful theme in seconds.

使用Hexo，主要是轻便简洁，支持Markdown书写

准备工作

Github账号
安装Node.js
安装Git

下载安装Hexo

选一个目录，新建文件夹作为项目目录，如 blog

生成静态页面

本地启动

在终端看到

打开浏览器访问 http://localhost:4000 ，恩…博客跑起来了

写文章

在博地博客文件夹source -> _post文件夹下看到新建的markdown文件
使用Markdown编辑器打开文件，写文章…（啊。。。超喜欢Markdown的书写方式
保存，然后本地发布

部署Github

新建Repository
仓库名必须是your_user_name.github.io的格式
修改_config.yml 文件，来进行关联

deploy:
    type: git
    repo: https://github.com/cataway/cataway.github.io.git
    branch: master

将项目部署到Github上面
执行

发布

用浏览器访问 http://cataway.github.io/ ，就可以看到博客部署在github上了
