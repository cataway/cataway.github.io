凌晨两点的深圳

发表于 2017-06-23 | 分类于 叙事高手

六月是雷雨的季节，特别是在南方沿海的城市。
出门从来不看黄历的我，选择了这样的一个日子去成都，吃火锅…
踩着点拖着行李箱到机场，发现人满为患，到处广播着飞机延误/取消的广播
我心想着，可以，总算赶上了，不用担心误机了
但是总感觉，火锅离我越来越远了…

坐在行李箱上拿着kindle继续《如果，宅》第二部
真遗憾没见过三季稻
作为一个大法爷居然没听说过三季稻，我这么被鄙视过…

等了一个多小时，等来了一个消息，我们要被拉去酒店休息了…
是的，今天晚上我是吃不上火锅了，我这么安慰自己
一趟大巴到酒店，认识了一个香港的姐姐，住一间房
看着很年轻，特别是手，完全不像一个中年妇女。她说她大女儿跟我一样大了
聊得还挺开心的，最后还请我吃了饭…

半夜两点的时候，终于接到消息说，可以登机了，外面的雨也停了
我睡得昏昏沉沉的，蛋定得已经内心毫无波澜了
上飞机的时候，听说，已经有一部分人十二点就在飞机上等待起飞了
要不是有一个人打电话给航空公司，航空公司才知道还有一部分人被遗留在酒店，就是我们…
要不然…我们就真的被遗忘了
竟然还能出这种乌龙事件
很多人连骂街的力气都没有了，毕竟半夜了，也骂了一晚上了…

成都还算不错…
吃吃吃…在酒店点外卖…
对于一个路痴加吃货来说，这样的旅游也是不错的

晚上的灰机回深圳，一如既往延误，回来已经没有地铁了
打了个滴滴，滴滴司机有点胖，话很多…
说他零几年就在深圳了，当初万科八千块一平的时候没买，要不然现在就是千万富翁了
我感叹，身边坐着一个错过了千万的土豪…
深圳这几年真是魔幻，哪里又不是呢，大型魔幻现实主义的社会…
我看着导航，盯着即将没电的手机，终于快到了
突然滴滴司机靠路边一个刹车
说不紧张，是对我颜值的不自信啊…啊呸，我当时紧张死了
我盯着滴滴司机，他转过头来淡定地对我说，车胎爆了
要不美女，你走回去吧。滴滴司机下车看了下前胎
我说，大半夜的，我怎么走回去
滴滴司机手指一挥说，你家就在前面，拐个弯就到了
我说那行吧
于是我拖着行李箱，走在马路上，天空还飘着细雨
身后传来滴滴司机的喊声，美女，你真的走啦，不留下来陪下我嘛
我头也不回地…

到了十字路口，我站在路牌下想看方向
对面走来一对情侣，有说有笑，应该是刚看完夜场的电影
凭着直觉跟着他们后面走，于是，我走到了楼下
终于看到了熟悉的大门…

宅女出趟门，真不容易
躺在床上，感觉活着回来真好…
