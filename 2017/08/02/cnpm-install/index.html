淘宝npm镜像

发表于 2017-08-02 | 分类于 Work

npm(node package manager) 是一个基于nodejs的Javascript包管理工具。使用npm包管理工具，可以在项目中方便地引入第三方的框架或者库，比如jQuery/Bootstrap/React，前提是这些框架都托管在npm上。而不需要像以前那种要去框架的官网下载包再复制黏贴到自己的项目下再引用的开发方式。

但是由于天朝特色，直接使用npm可能会导致下载安装依赖包很慢，甚至会失败，出现一大堆的error 或者 warning，看着整个人都不好了。所以可以更换成国内淘宝npm镜像，即cnpm
